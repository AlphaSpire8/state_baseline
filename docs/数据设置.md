# 数据设置（全量数据）

## 下载数据集

```bash
modelscope download \
  --dataset arcinstitute/State-Tahoe-Filtered \
  --local_dir /datasets/fanpeishan/data/tahoe
```

## 原始数据路径

/datasets/fanpeishan/data/tahoe/*.h5ad (from c0 to c49)

## 基因表达谱情况

表达矩阵是在原始counts的基础上，做了sc.pp.normalize_total(adata,target_sum=4000)处理。

## 数据划分规则

要验证集：
1. 训练集(40个):c0-c3，c5-c16,c21,c25-c30,c32-c37,c39-c49
2. 验证集(5个):c4,c17,c18,c23,c38
3. 测试集(5个):c19,c20,c22,c24,c31

不要验证集：
1. 训练集(40个):c0-c3,c5-c16,c21,c25-c30,c32-c37,c39-c49
2. 测试集(10个):c4,c17-c20,c22-c24,c31,c38

## 数据预处理流程

准确的数据预处理流程见： /datasets/fanpeishan/data/tahoe_for_scgen/data_preprocess_pipeline.py。
处理流程的核心包括：精简 obs、删除 obsm、按 ST_tahoe_var_dims.pkl 的 gene_names 严格对齐 2000 基因、补充 drug_index 和 cell_type、再按 drug_index 拆分保存。

## 预处理后的数据

1. 位置：/datasets/fanpeishan/data/tahoe_for_scgen/split_by_drug
2. 概况：
```
/datasets/fanpeishan/data/tahoe_for_scgen/split_by_drug/
├── celltype_c0/
│   ├── celltype_c0_drugindex_0.h5ad
│   ├── ...
│   └── celltype_c0_drugindex_379.h5ad
├── ...
└── celltype_c49/
    ├── celltype_c49_drugindex_0.h5ad
    ├── ...
    └── celltype_c49_drugindex_379.h5ad
```
3. 标准命名形式为：celltype_c{cell_type}_drugindex_{drug_index}.h5ad，其中 cell_type ∈ {c0, c1, ..., c49}。。
4. 每一个.h5ad文件含有6个obs列：obs['drugname_drugconc']/obs['drug']/obs['cell_name']/obs['plate']/obs['drug_index']/obs['cell_type']。其中，obs['drug_index']列和obs['cell_type']列的值，和其文件名称中的drug_index/cell_type相同。
5. 每一个.h5ad文件的基因种类和顺序，与/datasets/fanpeishan/data/docs/ST_tahoe_var_dims.pkl文件的"gene_names"键对应的值一致。
6. 每个药物和drug_index的对应关系参考/datasets/fanpeishan/data/docs/drug_name_list.csv文件（一列，列名为drug_name）。
7. 全局固定规模：2000 个基因（种类/顺序都相同）、380 个药物（种类相同）、50 个 cell_type。
