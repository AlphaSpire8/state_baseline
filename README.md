# a baseline repo

## 项目任务

我们在STATE模型的基础上进行了创新，以增强其zero-shot任务的能力。

## 仓库介绍

本仓库在三个数据集上，复现了多个模型的结果，给出了一个baseline，用于之后和我们增强后的state模型进行对比。

本地开发，4090/A100使用。

## 三个数据集

- tahoe
- nips
- tahoe-small

## 模型

（tahoe/nips）

- baseControl
- baseMLP
- biolord
- scGen
- scVI

（tahoe-small）

- pca_ridge
- biolord
